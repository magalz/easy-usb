# easy-usb

Cross-platform USB/IP daemon and client — share USB devices over IP with zero config.

[![CI](https://github.com/easy-usb/easy-usb/actions/workflows/ci.yml/badge.svg)](https://github.com/easy-usb/easy-usb/actions/workflows/ci.yml)
[![License: GPL-3.0-or-later](https://img.shields.io/badge/license-GPL--3.0--or--later-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Rust: 1.85+](https://img.shields.io/badge/rust-1.85%2B-orange.svg)](https://www.rust-lang.org)

## Install

### Linux
```sh
cargo install easy-usb-daemon easy-usb-cli
```

### Windows
```sh
cargo install easy-usb-daemon easy-usb-cli
```

### Android
See [docs/android.md](docs/android.md).
